# 🧶 HANKED — warm & woven
> Turning strings into stories, one loop at a time.

**HANKED** is a boutique handmade crochet brand founded by **Noor & Amna**. We specialize in creating high-quality, hand-crocheted items that blend comfort with craftsmanship. From cozy accessories to custom-imagined pieces, every stitch is made with love.

## ✨ Features
- **Curated Collection:** Browse our latest handmade crochet pieces.
- **Bespoke Custom Orders:** A dedicated portal where users can upload reference images and describe their dream projects.
- **Smooth User Experience:** Fully responsive design with elegant transitions and a modern "warm" aesthetic.
- **Order Management:** Integrated with **Forminit** to handle custom requests and file uploads reliably.
- **Interactive Cart:** A slide-out drawer to manage selected items before checkout.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox/Grid), JavaScript (ES6+).
- **Typography:** Playfair Display (Serif) & Inter (Sans-serif) via Google Fonts.
- **Form Handling:** [Forminit](https://forminit.com/) (Standard POST submission for reliable image uploads).
- **Hosting:** [Vercel](https://vercel.com/) (Automated deployment via GitHub).
